# 🤖 AI Interview Coach

An AI-powered mock interview platform that helps students prepare for real-world interviews through voice interaction, realistic questions, and performance feedback.

---

## 🚀 Features

* 🎤 Voice-based interaction (Microphone support)
* 🧠 AI-generated interview questions
* 💬 Real-time conversation simulation
* 📊 Performance feedback system
* 🌐 Clean and responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **AI/ML:** NLP-based interaction
* **Other:** Web Speech API / Microphone Integration
